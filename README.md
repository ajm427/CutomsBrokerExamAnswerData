# Reference Frequency Analyzer

This project analyzes large blocks of exam answer keys and legal citation text (such as CBLE materials) and converts them into structured, ranked frequency data. It extracts meaningful authorities (CFR, HTSUS, GRI, General Notes, ACE/BRPD, etc.), normalizes inconsistent formats, and reports which sources appear most often.

The goal is to transform messy, unstructured citation text into clear, quantitative insight that can guide study priorities and reveal patterns in exam construction.
